### Fórmulas

1. **Fórmula general del coeficiente de correlación de Pearson**:

$$
r = \frac{\sum_{i=1}^{n} (x_i - \overline{x})(y_i - \overline{y})}{\sqrt{\sum_{i=1}^{n} (x_i - \overline{x})^2} \sqrt{\sum_{i=1}^{n} (y_i - \overline{y})^2}}
$$

2. **Cálculo de la media de x e y**:
   
$$
\overline{x} = \frac{\sum_{i=1}^{n} x_i}{n} \quad \text{y} \quad \overline{y} = \frac{\sum_{i=1}^{n} y_i}{n}
$$

**Donde**:
- $n$ es el número total de observaciones (5 días en este caso)
- $x_i$ representa el tiempo de traslado del día $i$
- $y_i$ representa el nivel de cansancio del día $i$
- $\overline{x}$ es la media del tiempo de traslado
- $\overline{y}$ es la media del nivel de cansancio

**Codigo en Scala:**
```scala

val dias = List (("Lunes",32,4),("Martes",33,2),("Miercoles",57,1),("Jueves",32,8),("Viernes",90,8))

//Cantidad de observaciones
val n = dias.length

//obtencion de la media de tiempos de traslado
val x = dias.map(_._2).sum / n.toDouble

//obtencion de la media de nivel de cansancio
val y = dias.map(_._3).sum / n.toDouble

// Formula del numerador
val rs1 = dias.map(p => (p._2-x)*(p._3-y)).sum.toDouble


val rs2s1 = dias.map(p=> Math.pow((p._2 - x),2)).sum.toDouble
val rs2s2 = dias.map(p=> Math.pow((p._3 - y),2)).sum.toDouble
val rs2 = Math.sqrt(rs2s1)*Math.sqrt(rs2s2).toDouble
val rTotal = rs1/rs2.toDouble
```

### Información para la interpretación de los resultados

> El coeficiente de correlación de Pearson (r) es una medida estadística que indica la fuerza y la dirección de la relación lineal entre dos variables. Es útil para determinar si un cambio en una variable se asocia con un cambio en la otra y qué tan fuerte es esa relación. Los valores de r van de -1 a 1.

#### Interpretación del coeficiente de correlación:

| Valor de r | Interpretación |
|------------|----------------|
| **r = 1** | Correlación positiva perfecta: a mayor tiempo de traslado, mayor cansancio de forma proporcional |
| **0.7 ≤ r < 1** | Correlación positiva fuerte: más tiempo de traslado está claramente asociado con mayor cansancio |
| **0.4 ≤ r < 0.7** | Correlación positiva moderada: existe cierta relación entre más tiempo de traslado y más cansancio |
| **0.1 ≤ r < 0.4** | Correlación positiva débil: la relación entre tiempo de traslado y cansancio es mínima |
| **r = 0** | No hay correlación lineal: el tiempo de traslado no predice el nivel de cansancio |
| **-0.4 < r ≤ -0.1** | Correlación negativa débil: más tiempo de traslado se asocia levemente con menos cansancio |
| **-0.7 < r ≤ -0.4** | Correlación negativa moderada: existe relación inversa entre tiempo de traslado y cansancio |
| **-1 < r ≤ -0.7** | Correlación negativa fuerte: más tiempo de traslado está claramente asociado con menor cansancio |
| **r = -1** | Correlación negativa perfecta: a mayor tiempo de traslado, menor cansancio de forma proporcional |

**3. Explique detalladamente lo que sucede en la línea de código donde se calcula el numerador y el denominador de la fórmula del coeficiente de correlación de Pearson. ¿Qué representa cada suma y cada operación?**
   A. El Numerador:
```scala
val numerador = pares.map { case (x, y) => (x - mediaX) * (y - mediaY) }.sum
```
Esta línea calcula la covarianza, es decir, si las variables "se mueven juntas".

pares.map recorre cada día (cada par de datos (x, y)).

(x - mediaX) calcula si el tiempo de ESE día fue mayor o menor que el promedio de tiempo.

(y - mediaY) calcula si el cansancio de ESE día fue mayor o menor que el promedio de cansancio.

Al multiplicarlos, si ambos son positivos (o ambos negativos), da un número positivo. Si uno sube y el otro baja (como en mi caso), da un número negativo.

.sum suma todos esos resultados para ver la tendencia general.

B. El Denominador:
```scala
val denominador = Math.sqrt(sumaX) * Math.sqrt(sumaY)
```
Esta línea mide la variabilidad total de los datos; sirve para "normalizar" el resultado.

sumaX (que viene de dataX.map(x=>Math.pow(x-mediaX,2)).sum) mide cuánto variaron mis tiempos entre sí, sumando sus desviaciones al cuadrado.

sumaY (que viene de dataY.map(y=>Math.pow(y-mediaY,2)).sum) mide cuánto varió mi cansancio entre sí.

Math.sqrt saca la desviación estándar de cada variable.

Al multiplicar ambas desviaciones estándar, obtenemos un valor que escala el numerador, asegurando que el resultado r siempre quede entre -1 y 1.
